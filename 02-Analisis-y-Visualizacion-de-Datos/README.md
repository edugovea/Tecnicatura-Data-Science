# Análisis y Visualización de Datos 
 
## Proyecto Final: Dashboard Exportaciones Argentinas API 
 
Dashboard interactivo con datos reales de exportaciones argentinas (2020-2025), obtenido vía API INDEC. 
Incluye 4 visualizaciones: composición por provincia, evolución temporal, mapa geográfico y productos primarios.
